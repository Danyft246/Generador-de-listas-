Gestor de Listas Enlazadas

Aplicación web interactiva para aprender y practicar **Listas Enlazadas** (Simple, Doble y Circular).
Características

Interfaz moderna con pestañas
Lista Simple, Lista Doble y Lista Circular
Operaciones completas:
  - Insertar al inicio y al final
  - Eliminar primero, último y por valor
  - Buscar elemento (identifica si el valor esta presente o no)
  - Mostrar lista inversa (en lista doble)
Tecnologías usadas
Backend: Python + Flask
Frontend: HTML5, CSS3 y JavaScript (Vanilla)
Estructuras de datos: Implementadas desde cero

Cómo ejecutar el proyecto

bash
Clonar el repositorio
git clone https://github.com/Danyft246/Generador-de-listas-.git
Entrar a la carpeta
cd Generador-de-listas-
Instalar dependencias
pip install -r requirements.txt
Ejecutar la aplicación
python app.py
